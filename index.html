Bienvenidos a JBlog!

El presente blog intenta cubrir temas relacionados con Java y sus frameworks subyacentes. También se incluyen algunas temáticas generales como por ejemplo la tabla ASCII o las expresiones regulares.

Hay artículos extensos y otros no tanto. Algunos escritos en español y otros en inglés. Siempre que se tome información de otra fuente habrá un link o cita a la misma.

El sitio completo esta hosteado en Github así que pueden clonarlo y/o forkearlo. Fue creado con Jekyll que es una herramienta generadora de sitios estáticos (por defecto es la usada en Github).


Sugerencias, contribuciones y comentarios serán altamente apreciados!
