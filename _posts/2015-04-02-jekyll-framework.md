## What is Jekyll?

[Jekyll](http://jekyllrb.com) is a static site generator, an open source tool to create simple yet powerful websites of any size and scale. You can checkout the code from [Github](https://github.com/jekyll/jekyll). This framework is based on the Ruby language.

Wikipedia [describes it as](https://en.wikipedia.org/wiki/Jekyll_(software)):

> Jekyll is a simple, blog-aware, static site generator. It takes a template directory containing raw text files in various formats, runs it through a converter (like Markdown) and our Liquid renderer, and spits out a complete, ready-to-publish static website suitable for serving with your favorite web server. Jekyll also happens to be the engine behind GitHub Pages, which means you can use Jekyll to host your project’s page, blog, or website from GitHub’s servers for free.

## Key features

- It gives the blogger total control of the site. Some solutions out there are not too flexible as Jekyll.
- Totally compatible with [Github Pages](https://pages.github.com/). Push your posts and Github automatically generates the site in a matter of seconds. If something goes wrong with the generation phase, Github sends you an email describing the error.
- Completely free if used with Github!
- You can define your files in a standard format like [Markdown](https://en.wikipedia.org/wiki/Markdown) and Jekyll takes the responsibility of read those files and generate HTML ready to use.
